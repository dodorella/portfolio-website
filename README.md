# Personal Portfolio

A personal portfolio website showcasing my projects, skills and background as a Mobile Software Development student.

## About

This portfolio was built to present my work, technical skills and development experience in a clean and modern way. It highlights selected projects, provides information about my background and offers ways to get in touch.

## Features

* Responsive design
* Modern glassmorphism-inspired UI
* About Me section
* Skills overview
* Project showcase
* Contact section with GitHub, LinkedIn and Email links
* Downloadable CV

## Technologies Used

* HTML5
* CSS3
* JavaScript


## Featured Projects

### Save The Forest

A 2D game developed with LibGDX where players protect the forest by overcoming obstacles and surviving as long as possible.

### Smart Task Manager

A modern task management application featuring local storage and task prioritization.

More projects can be found on my GitHub profile.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/dodorella/portfolio.git
```

Open `index.html` in your browser or run the project using a local development server.
